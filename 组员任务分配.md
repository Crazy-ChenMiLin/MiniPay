# MiniPay 组员任务分配

## 核心链路

创建订单 → 发起支付 → 支付状态更新 → 结果查询

---

## 后端任务

### 订单服务（order-service）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/orders` | POST | 创建订单，生成 orderId，存库，返回订单信息 |
| `/api/orders/{orderId}` | GET | 查询订单，返回订单状态 |
| `/api/orders/{orderId}/status` | PUT | 更新订单状态（供支付服务回调），状态：PENDING → PAID/FAILED |
| `/api/orders/health` | GET | 健康检查，返回 OK |

### 支付服务（payment-service）

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/payments` | POST | 发起支付，创建支付记录，模拟支付结果（80%成功/20%失败） |
| `/api/payments/{orderId}` | GET | 根据 orderId 查询支付状态 |
| `/api/payments/health` | GET | 健康检查，返回 OK |

### 服务间通信

支付完成后，支付服务通过 HTTP 调用订单服务更新订单状态：

```
PUT http://order-service:8081/api/orders/{orderId}/status
请求：{ "status": "PAID" }  或  { "status": "FAILED" }
```

### 接口请求/响应格式

```
POST /api/orders
请求：{ "amount": 100.00, "description": "测试订单" }
响应：{ "orderId": "202401010001", "amount": 100.00, "status": "PENDING", "createdAt": "..." }

GET /api/orders/{orderId}
响应：{ "orderId": "202401010001", "amount": 100.00, "status": "PAID", "description": "...", "createdAt": "..." }

PUT /api/orders/{orderId}/status
请求：{ "status": "PAID" }
响应：{ "orderId": "202401010001", "status": "PAID" }

POST /api/payments
请求：{ "orderId": "202401010001", "amount": 100.00 }
响应：{ "paymentId": "P001", "orderId": "202401010001", "status": "SUCCESS", "paidAt": "..." }

GET /api/payments/{orderId}
响应：{ "paymentId": "P001", "orderId": "202401010001", "status": "SUCCESS", "amount": 100.00, "paidAt": "..." }
```

## 前端任务

### 页面

| 页面 | 路由 | 功能 |
|------|------|------|
| 创建订单 | `/` | 输入金额 + 描述，点击创建，跳转到支付页 |
| 发起支付 | `/pay/:orderId` | 展示订单信息，点击支付按钮，显示支付结果 |
| 结果查询 | `/query` | 输入 orderId，查询并展示订单状态和支付状态 |

### 基础设施

- 恢复 vue-router，配置3个页面路由
- 顶部导航栏，页面间切换
- 封装 API 调用

### API 封装

```javascript
// api/index.js
export const createOrder = (data) => api.post('/orders', data)
export const getOrder = (orderId) => api.get(`/orders/${orderId}`)
export const createPayment = (data) => api.post('/payments', data)
export const getPayment = (orderId) => api.get(`/payments/${orderId}`)
```

### 涉及文件

- `frontend/src/router/index.js` — 新建，配置路由
- `frontend/src/main.js` — 注册 router
- `frontend/src/App.vue` — 导航 + router-view
- `frontend/src/api/index.js` — 封装 API
- `frontend/src/views/CreateOrder.vue` — 创建订单页
- `frontend/src/views/PayOrder.vue` — 发起支付页
- `frontend/src/views/QueryResult.vue` — 结果查询页

## 协作规范

- 前端通过网关（localhost:8080）调用后端，不要直连服务
- 本地启动需配置数据库密码（环境变量 `MYSQL_PASSWORD` 或直接改 application.yml）
- 提交前先 `git pull` 拉取最新代码，避免冲突
